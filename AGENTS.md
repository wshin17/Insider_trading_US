# AGENTS

## Working rules

- Treat this repository as a US-first research project starting from zero on data collection.
- Read files under `docs/context/` before proposing code or architecture changes.
- Prioritize data collection strategy before implementation.
- Do not assume real data already exists.
- Do not hard-code local or server paths unless explicitly requested.
- Do not import assumptions from the older non-US project unless explicitly requested.
- Keep changes small, explicit, and reviewable.
- Use `literature/notes/` as the primary reusable literature knowledge base.
- At the end of each substantial task, summarize files changed, commands run, and open questions.
