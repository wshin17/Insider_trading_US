# Insider_trading_US

This repository is a clean US-market research project focused on insider trading, executive selling behavior, SEC filings, and related multimodal evidence.

## Current stage

This project starts from zero on the data side.

The immediate priority is:
1. define the US data collection strategy
2. define the source registry
3. define key identifiers and event ontology
4. prepare for later real-data onboarding

## Near-term focus

- collect and organize US-market data sources
- study SEC-centered disclosure and transaction data
- build a reusable literature and method knowledge base
- delay implementation details until the data strategy is clearer
